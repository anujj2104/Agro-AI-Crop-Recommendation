# Agro-AI-Crop-Recommendation
ML-based crop recommendation system using Random Forest and GUI.
This project uses Machine Learning (Random Forest) to recommend the best crop based on soil and environmental conditions.

## Features
- Predicts best crop using N, P, K, temperature, humidity, pH, rainfall
- GUI built using CustomTkinter
- Visualization using Matplotlib

## How to Run
pip install customtkinter, pandas, numpy, matplotlib, scikit-learn  
python miniproject.py
